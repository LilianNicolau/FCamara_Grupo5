## 🖋 Descrição do projeto

O projeto *Lista do Bem* tem como objetivo criar uma aplicação onde pais irão cadastrar seus filhos que estudam em escolas públicas e os materiais que eles precisam para cursar aquele ano letivo. 
Também pede-se que usuários avulsos possam acessar a aplicação, conhecer os alunos que precisam desses materiais e fazer adoação dos itens para esse aluno. 
O que devemos entregar?
- uma aplicação responsiva;
- mínimo de três telas;
